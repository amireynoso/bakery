Brief history of the site

I launched this site back in the (northern hemisphere) summer of 2020. I originally built it from scratch using Jekyll with a completely custom theme and was using a deploy process built with GitHub actions. However, writing in Markdown and everything that was involved in publishing recipes drove me to not really post anymore.

In July 2021, I came across Publii and decided to give it a go. Yes, I decided to take off the developer hat and use a pre-built CMS solution and theme. No regrets. My main focus is on the content.

Because I have a hard time letting go, all the source code from the original version can still be found on this branch.
